## 本次优化方向

- 将原页面的深色 agency 结构切换为更接近 Spotify 的沉浸式黑绿视觉系统
- 统一按钮、标签、导航为 pill / circle 几何语言
- 把绿色控制在 CTA、active、play 等功能性位置，避免装饰性滥用
- 压缩层级和间距，增强首页的信息密度和“产品界面感”
- 重做 hero 右侧主视觉，使其更像音频播放器面板而不是一般展示卡片
