# figmaAutoDesign

A minimal TypeScript starter project for automating design-related workflows around Figma.

## Quick start

```bash
npm install
npm run dev
```

## Available scripts

- `npm run dev`: run the local entry file with `tsx`
- `npm run build`: compile TypeScript to `dist`
- `npm start`: run the compiled output

## Environment

Copy `.env.example` to `.env` when you are ready to add real credentials.

## GitHub

This repository includes a basic GitHub Actions workflow that installs dependencies and runs the TypeScript build on every push and pull request.
