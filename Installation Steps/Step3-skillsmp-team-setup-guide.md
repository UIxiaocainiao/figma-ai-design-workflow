## 可直接复制的团队介绍文案

我们要创建一个 2 人专业团队：

- 专业 UI 设计师：负责信息架构、页面设计、视觉规范、设计系统、组件规范、交互方案、Figma 设计交付与开发协作。
- 专业前端开发工程师：负责 React / Next.js 前端实现，具备丰富动画制作经验，熟悉 GSAP、ScrollTrigger、Lenis、Framer Motion、滚动视差、页面转场和高级微交互，能够将设计高质量还原为可上线的产品页面。

---

## 可直接复制的团队要求文案

团队成员需要具备以下能力：

1. 能完成高质量 UI/UX 设计与设计系统搭建。
2. 能输出清晰的页面结构、视觉规范和组件规范。
3. 能基于 Figma 完成交付，并与开发高效协作。
4. 能使用 React / Next.js / Tailwind 实现高还原度页面。
5. 能独立完成高级网页动画，包括滚动动画、视差、转场、微交互和沉浸式视觉效果。
6. 注重审美、性能、响应式适配与用户体验。

---

## 一句话版本

我们需要一个专注于 UI 设计 + 动效前端开发 的双人团队：
一人负责专业 UI/UX 与设计系统，一人负责高质量前端实现与高级网页动画制作。
